## 1. WHY — Tại sao cần học

Bạn đã master Claude Code. Nhưng impact limited nếu chỉ mình bạn biết.

Teaching multiply value: team productive hơn, bạn consolidate knowledge, trở thành expert, organization benefit compound. Module này equip bạn share hiệu quả — từ quick tip đến full workshop.

---

## 2. CONCEPT — Khái niệm cốt lõi

### Teaching Format

| Format | Duration | Best For |
|--------|----------|----------|
| **Tips Session** | 15-30 min | Quick win, demo |
| **Lunch & Learn** | 45-60 min | Single topic |
| **Half-day Workshop** | 3-4 giờ | Hands-on practice |
| **Full-day Training** | 6-8 giờ | Comprehensive |

### Design Principle

1. **Show, Don't Tell** — Live demo hơn slide
2. **Hands-On Practice** — 50% lecture, 50% practice
3. **Progressive Complexity** — Quick win trước advanced
4. **Relevance** — Dùng code của họ
5. **Takeaway** — Cheat sheet và template

### Workshop Structure

```text
Hook (10%) → Foundation (20%) → Main (50%) → Advanced (20%) → Wrap (10%)
```

## 3. DEMO — Từng bước cụ thể

### Workshop "Claude Code Kickstart" 2 Giờ

**Audience**: Team mới dùng Claude Code
**Goal**: Basic proficiency, productive ngay

**Schedule**:

**0:00-0:15 — HOOK**
- Live demo: Build feature trong 5 phút
- "Cuối buổi, bạn sẽ làm được như này"

**0:15-0:35 — FOUNDATION**
- Claude Code là gì? (5 phút)
- Kiểm tra installation (5 phút)
- Lệnh đầu tiên cùng nhau (10 phút)

**0:35-1:05 — CORE TECHNIQUE**
- CLAUDE.md introduction (10 phút)
- Hands-on: Viết CLAUDE.md (15 phút)
- Share và compare (5 phút)

**1:05-1:15 — BREAK**

**1:15-1:45 — PRACTICAL WORKFLOW**
- Live demo: Read + write code (20 phút)
- Hands-on: Complete một task (10 phút)

**1:45-2:00 — WRAP-UP**
- Recap 3 technique chính
- Phát cheat sheet + template
- Q&A

### Demo Script Template

```markdown
# Demo: [Tên]
Duration: [X] phút

## Setup (trước workshop)
- [ ] Project đã clone
- [ ] CLAUDE.md sẵn sàng
- [ ] Command đã test

## Script
1. "Để tôi show [capability]"
2. Gõ: `claude "[command]"`
3. Giải thích output
4. "Giờ đến lượt bạn..."

## Backup
Nếu demo fail: [phương án thay thế]
```

## 5. CHEAT SHEET

### Workshop Structure

| Phase | Time | Focus |
|-------|------|-------|
| Hook | 10% | Demo ấn tượng |
| Foundation | 20% | Core concept |
| Main | 50% | Technique + practice |
| Advanced | 20% | Scenario phức tạp |
| Wrap | 10% | Recap + resource |

### Time Allocation

- 50% Hands-on practice
- 30% Live demo
- 20% Lecture

### Material Checklist

- [ ] Demo script (đã test!)

## 7. REAL CASE — Câu chuyện thực tế

**Scenario**: Tech company Việt Nam rollout Claude Code cho 50 developer, 5 team.

**Teaching Strategy**:
- **Tuần 1**: Train-the-Trainer — 4 senior dev thành champion
- **Tuần 2-3**: Mỗi champion run 4-hour workshop cho team
- **Tuần 4**: Office hour, Slack channel, monthly advanced session

**Material**:
- 2-hour kickstart workshop (reusable)
- Team-specific CLAUDE.md template
- Cheat sheet laminated trên mỗi bàn

**Kết quả (2 tháng)**:
- 50/50 developer active dùng
- 3 champion thành internal expert
- New hire onboarding có workshop
- Productivity: +30%

**Quote**: "Không chỉ dạy tool. Tạo internal expert tiếp tục dạy người khác. Đó là cách knowledge scale."
