## MangalaHQ

Chúng tôi là một đội nhỏ các developer, xây dựng công cụ giúp phát triển phần mềm nhanh hơn và đáng tin cậy hơn.

**ShipWithAI** là hệ sinh thái plugin cho Claude Code. Chúng tôi tự build plugin — Auth, Payment, Deploy CI/CD — dùng cho chính sản phẩm của mình, rồi chia sẻ với cộng đồng developer.

## Triết lý

- **Dogfooding first**: Mọi plugin đều được thử nghiệm trên production app thật trước khi release
- **Developer experience là ưu tiên số 1**: Nếu khó dùng thì chưa xong
- **Open ecosystem**: Chúng tôi xây platform, cộng đồng xây plugin

## Sản phẩm

- **ShipWithAI Plugins**: Auth, Payment, Deploy CI/CD cho Claude Code
- **Claude Code Mastery**: Khóa học toàn diện cho developer
- **Toolkit Reviews**: Đánh giá trung thực các công cụ AI coding
