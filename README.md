## What You'll Learn

From zero to production-ready Claude Code workflows:

- **Foundation** — Installation, interfaces, context window
- **Security** — Threat modeling, sandboxing, secret management
- **Core Workflows** — Reading code, writing code, git, shell
- **Prompt Engineering** — CLAUDE.md, slash commands, memory
- **Context Mastery** — Optimization, visual context
- **Thinking & Planning** — Extended thinking, plan mode
- **Multi-Agent** — Auto coding, agentic loops, orchestration
- **Meta-Debugging** — Hallucination detection, loop breaking
- **Legacy Code** — Archeology, refactoring, tech debt
- **Team Collaboration** — Team CLAUDE.md, code review, governance
- **Automation** — Headless mode, SDK, hooks, GitHub Actions, MCP
- **n8n Integration** — Workflow patterns, orchestration
- **Data & Analysis** — Reports, log analysis
- **Optimization** — Speed, quality, cost
- **Templates & Skills** — CLAUDE.md templates, prompt recipes
- **Real-World Mastery** — Case studies, role-specific workflows

## Who This Is For

Senior developers who want to go beyond basic Claude Code usage. You should already know how to code — this course teaches you how to make Claude Code work for you at a professional level.
